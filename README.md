# 🥕 당근코 – 지역 기반 러닝 코스 커뮤니티 서비스

> **“어디서 뛰어야 할지 모르는 초보 러너들을 위한 로컬 러닝 코스 공유 플랫폼”**  
> 당근코는 **동네 기반 러닝 코스 추천**과 **경험 공유 커뮤니티**를 통해  
> 부담 없이 러닝을 시작하고 지속할 수 있도록 돕는 서비스입니다.

---

## ✨ 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 당근코 (당신 근처의 코스) |
| **목적** | 초보 러너의 진입장벽을 낮추고, 지역 커뮤니티를 활성화 |
| **기간** | 2025.11 ~ 2025.12 |
| **참여자** | 주세빈, 양동근 |
| **기술 스택** | Spring Boot · MySQL · (추가 예정) |

---

## 🧍 Persona – 금쪽같은 동근이

| 프로필 | 내용 |
|--------|------|
| **양초보 (25세, 남)** | 러닝 입문자 |
| **Needs** | 어디서 뛰어야 할지 모름, 초보자 코스 정보 필요 |
| **Pain Points** | 러닝 앱은 기록 중심 / 코스 정보는 유명지 편중 / 커뮤니티 부재 |
| **선호 코스** | 짧고 편한 동네 코스, 한적한 곳, 러닝크루 불편 |

---

## 🎯 목표 (Objectives & Key Results)

| 목표 | 지표 |
|------|------|
| **코스 데이터 확보** | 론칭 6개월 내 등록 코스 1,000개 |
| **커뮤니티 활성화** | MAU 대비 리뷰 작성률 10% |
| **지역 커버리지 확보** | 사용자 위치 기준 1km 내 검증된 코스 90% |

---

## 🚩 서비스 핵심 가치

### 🔍 Solution
- 초보 러너를 위한 **코스 선택 중심 UI**
- **현장성 있는 경험 공유 커뮤니티**
- **지역 기반 러닝 데이터베이스 구축**

### 😣 Problem
- 기존 러닝 앱은 **기록 중심** → 초보자 니즈 미해결
- 유명 명소 중심의 **편중된 코스 정보**
- **초보자 커뮤니티 부재**
